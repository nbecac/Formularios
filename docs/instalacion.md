# Instalación en Windows

1. Asegúrate de tener Python instalado y accesible en tu PATH.
2. Abre una terminal (CMD o PowerShell) y ve a la ruta:
   `C:\Users\Nicolás\Desktop\Formularios`
3. Ejecuta `scripts\setup.bat`. Esto creará la carpeta `.venv` e instalará FastAPI, SQLite, etc., además de poblar la base de datos.
4. Carga la extensión en Chrome a través de `chrome://extensions` activando el modo desarrollador.
5. Usa `scripts\run_all.bat` para iniciar el servidor local.
